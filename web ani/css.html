CSS Scale

CSS Scale modifies the size of an element, enlarging or reducing it proportionally.



CSS Skew

CSS Skew tilts or slants an element along the horizontal or vertical axis, creating a diagonal transformation.



CSS Translate

CSS Translate shifts an element from its original position on the horizontal and/or vertical axis, enabling movement.



CSS Rotate

CSS Rotate rotates an element clockwise or counterclockwise by a specified degree, altering its orientation.
